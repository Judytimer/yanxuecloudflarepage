# 字体文件说明

## 华文彩云（当前使用）

当前使用的治愈风格字体：**华文彩云**

### 字体文件：
- `华文彩云.ttf` - 已配置为默认字体

### 字体特点：
- 治愈风格的彩云字体
- 适合家庭教育类应用
- 温暖、友好的视觉感受
- 具有艺术感和装饰性

## 新蒂剪纸体（备用）

### 字体文件：
- `新蒂剪纸体.ttf` - 已配置为备用字体

### 字体特点：
- 治愈风格的手写字体
- 适合家庭教育类应用
- 温暖、友好的视觉感受

### 系统备用字体：
如果字体文件加载失败，系统会自动使用：
- PingFang SC（苹果系统）
- Hiragino Sans GB（日文系统）
- Microsoft YaHei（Windows系统）
- WenQuanYi Micro Hei（Linux系统）
